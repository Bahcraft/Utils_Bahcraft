# BahCraft Utils

A Minecraft 1.20.1 mod that adds the magical items.

## Requirements

- Minecraft: 1.20.1
- Forge
- GeckoLib \(Required for animations\)
- AzureLib \(Required for pathfinding\)

## Installation

1. Download the mod jar from the releases section below.
2. Download the latest versions of GeckoLib and AzureLib compatible with Minecraft 1.20.1.
3. Put all three files (the mod jar, GeckoLib jar, and AzureLib jar) into your `mods` folder.
4. Launch Minecraft with the matching Forge profile.

## Usage

- Right\-click to cast the wand's current mode.
- Sneak + Right\-click to cycle modes. The item name displays on mode change.

## Notes

- GeckoLib is required for the wand animations; ensure the GeckoLib build targets 1.20.1.
- AzureLib is required for any pathfinding or AI utilities used by the mod.
